#  AI-Powered YouTube Summarizer

A web-app that generates yt video summaries from yt-transcript-api 's transcripts using model **openai/gpt-oss-120b**


##  Tech Stack
   
 Frontend:   HTML, CSS, JavaScript     
 Backend :    Python, Flask, Flask-CORS    
 LLM     :   openai/GPT-oss-120b on Groq     
 Transcript:  youtube-transcript-api    
 Deployment: Render (free tier)
